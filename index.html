ScanCode Workbench Help

Overview

ScanCode Workbench allows you take the scan results from the ScanCode and create a conclusion-level software inventory. The conclusions (Concluded License, Concluded Owner, etc.) you make when creating your conclusion can be exported as a JSON file or saved as SQLite file. For more information, see our wiki.
